# Landing Page Profissional (PHP)

Landing page simples, limpa e profissional para **qualquer profissional autônomo**.

## Características
- Apenas **2 arquivos**: `index.php` + `config.php`
- **PHP 8.3** | **Tailwind CSS (CDN)**
- 100% configurável via `config.php`
- Design sério, responsivo e moderno
- Pronto para produção

## Como usar
1. Edite `config.php` com seus dados
2. Suba para seu servidor PHP
3. Acesse!

## Personalização
- Troque nome, profissão, serviços, contatos
- Adicione foto (futuro: `/assets/foto.jpg`)
